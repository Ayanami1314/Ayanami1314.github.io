9.14 жёёжҲҸд»Јз Ғи®ҫи®Ўе“ІеӯҰпјҡ

жң¬иҙЁйңҖжұӮпјҡи§ЈиҖҰеҗҲгҖҒйҷҚдҪҺеӨҚжқӮеәҰ

и®ҫи®ЎжЁЎејҸпјҡ

и§ӮеҜҹиҖ…жЁЎејҸпјҡ

еҰӮжһңиҰҒи®ҫи®ЎдёҖдёӘвҖңиҗҪж°ҙвҖқзҡ„е®Ңж•ҙж•ҲжһңпјҢеҢ…еҗ«еЈ°йҹігҖҒзү№ж•ҲгҖҒзү©зҗҶзӯүпјҢеҰӮдҪ•и§ЈиҖҰжЁЎеқ—пјҹ

зӯ”

е°Ҷзү©зҗҶжЁЎеқ—жң¬иә«жҠҪеҮәпјҢйҳІжӯўзү©зҗҶжЁЎеһӢе’ҢUIж•ҲжһңиҖҰеҗҲ зү©зҗҶжЁЎеқ—еӨ„зҗҶе®ҢиҗҪж°ҙдәӢд»¶д№ӢеҗҺдә§з”ҹдёҖдёӘevent

и®©йҹійҮҸжЁЎеқ—гҖҒзү№ж•ҲжЁЎеқ—зӯүдҪңдёәдёҚеҗҢзҡ„и§ӮеҜҹиҖ…observerпјҢеҪ“жЈҖжөӢеҲ°eventдә§з”ҹпјҢжү§иЎҢеҜ№еә”еӨ„зҗҶд»Јз ҒгҖӮ

иҝҪй—®пјҡи®ҫи®Ўеә”иҜҘжҳҜеҗҢжӯҘзҡ„иҝҳжҳҜејӮжӯҘзҡ„пјҹд»Җд№Ҳжғ…еҶөдёҚеә”иҜҘдҪҝз”ЁиҝҷдёӘжЁЎејҸпјҹ

зҠ¶жҖҒжЁЎејҸпјҡе–„з”ЁзҠ¶жҖҒжңә

еҰӮжһңжңүиҝҷж ·дёҖдёӘйҖ»иҫ‘пјҡжҢүspaceи·іи·ғпјҢз©әдёӯе…Ғи®ёдәҢж®өи·іпјҢз©әдёӯжҢүJдёӢдҝҜеҶІж”»еҮ»пјҢдҝҜеҶІгҖҒи·іи·ғгҖҒиҗҪең°еқҮжңүеҠЁз”»пјҢеҰӮдҪ•и®ҫи®Ўпјҹ

зӯ”
зҠ¶жҖҒжңәпјҡ
       
дәәзү©зҠ¶жҖҒпјҡз«ҷз«ӢгҖҒз©әдёӯ1ж®өгҖҒз©әдёӯ2ж®өгҖҒиҗҪең°

       
ж”№еҸҳзҠ¶жҖҒзҡ„actпјҡз«ҷз«Ӣ->и·іи·ғ->з©әдёӯ1ж®өпјҢз©әдёӯ1/2->дҝҜеҶІпјҢз©әдёӯ1/2->жӯЈеёёиҗҪең°пјҢз©әдёӯ1->и·іи·ғ->з©әдёӯ2

       
дёҚж”№еҸҳзҠ¶жҖҒзҡ„actпјҡе…¶д»–
еҰӮжһңдёҚе®ҡд№үзҠ¶жҖҒжңәпјҢдҪ йңҖиҰҒи®°еҪ•зҡ„вҖңзҠ¶жҖҒ"жңүпјҡ жҳҜеҗҰз©әдёӯгҖҒжҳҜеҗҰиҗҪең°гҖҒжҳҜеҗҰдҝҜеҶІгҖҒи·іи·ғж®өж•°......

еҪ“зҠ¶жҖҒжӣҙеӨҡзҡ„ж—¶еҖҷпјҢйҖ»иҫ‘дјҡеҫҲеӨҚжқӮ

еҒҮи®ҫе·Із»ҸеӯҳеңЁact inputhandler(...){}еҮҪж•°пјҢ return act

еҲҷеҸӘйңҖе®ҡд№үиҝҷж ·зҡ„еҮҪж•°еңЁдё»еҫӘзҺҜдёӯжү§иЎҢ

           }
       }
