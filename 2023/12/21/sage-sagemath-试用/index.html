2023/12/21

sage(sagemath)试用

最好用的几个功能：

帮助sqrt?， 加一个?就行

tab 自动补全
转 latex 表达式，在表达式的外面套一层latex()即可

近似值：

x=sqrt(3)
n(x)  # 加上n()就是近似值

微分：

var('x')
diff(sin(x), x)
# 也可以偏导

积分：

#不定积分
var('x')
integral(x^2*cos(x), x)
#定积分
integral(x^2*cos(x), x, 0, 1)

解方程:

#精确解
var('x')
solve([x^3+x+6==5], x)
#近似解
f = x^3+8 == 10x
find_root(f,-1,2)

积分变换

s = var("s")
t = var("t")
f = t^2*exp(t) - sin(t)
f.laplace(t,s)

f2=1/(s^3+s)
inverse_laplace(f2,s,t)

微分方程

t = var('t')    # define a variable t
x = function('x')(t)   # define x to be a function of that variable
DE = diff(x, t) + x - 1
desolve(DE, [x,t])
(_C + e^t)*e^(-t)

画图

t = var('t')
sage: p1 = plot(cos(2*t) + 2*cos(t), (t,0, 2*pi), rgbcolor=hue(0.3))
sage: p2 = plot(4*cos(t) - cos(2*t), (t,0, 2*pi), rgbcolor=hue(0.6))
sage: show(p1 + p2)

特殊函数

x = polygen(QQ, 'x')
