  1. shuffle (first, last)
    Rearranges the elements in the range [first,last) randomly

  2. rotate(first, middle, end)

  3. sort系

  4. partition系
    patition(first, last, cond)
    重构顺序为，所有满足cond的元素在前，不满足cond的元素在后
    返回第一个不满足cond的元素的iter

  5. heap系

  • make_heap(first, last, comp)
    由于make_heap没有返回值是void(没有std::heap)，所以heap系都需要first, last参数,默认最大堆
  • sort_heap(first, last, comp)
  • pop_heap(first, last)
  • push_heap(first, last)
    (或者用priority_queue)

gdb tui:

输入tui enable或 ctrl-X A启动, tui disable关闭

当出现错误时，输入control + L 刷新屏幕

Ctrl-X 2：切换到汇编窗格

tui reg all:看寄存器
