2023/10/16 入门 命令行

linux 命令拓展

  • whatis: 简述man

  • whereis:列出所有， which 是一个

  • curl [url] > somefile：从url下载到somefile

  • find [root dir] -name “name(可包含通配符)”

​ eg find ./ -name “prefix*”

  • find . -perm /a=x 找到可执行文件

  • chmod +x [filename]让此文件可执行

  • ip address | grep eth0: 只看以太网0的ip地址

  • 看dns：resolvectl status

  • ping [website, eg www.google.com] ctl+c 退出

  • free：看内存

  • df -H: 看可用空间

  • ps -aux：看详细进程

  • ps -aux | grep [programName] 找到pid，然后可以kill pid或者 pkill -f [program name]

  • time ./a.out计时
